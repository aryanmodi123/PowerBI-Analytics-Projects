# Electricity Consumption Analytics Dashboard

## Objective
To analyze electricity consumption patterns and visualize usage trends using Power BI.

## Dataset
Electricity usage data including consumption units, time periods, and categories.

## Tools Used
- Power BI

## Key Insights
- Consumption trends over time
- High and low usage periods
- Comparative analysis across categories

## Outcome
Created an interactive Power BI dashboard to support monitoring and reporting of electricity usage.
