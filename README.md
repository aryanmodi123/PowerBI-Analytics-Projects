 Power BI Analytics Projects

This repository contains **Power BI analytics dashboards** created using real-world datasets.  
The projects focus on **reporting, performance analysis, and data visualization**, aligned with **Junior Data Analyst / Reporting Analyst** roles.

---

 Projects Included

 1. Ecommerce Sales Dashboard
Analyzes ecommerce sales performance using interactive Power BI visuals.
- Sales and revenue trends
- Category-wise and regional analysis
- KPI tracking for reporting

---

 2. Electricity Consumption Analytics Dashboard
Focuses on analyzing electricity usage patterns over time.
- Consumption trend analysis
- Identification of high and low usage periods
- Comparative analysis across categories

---

 3. Football Analytics Dashboard
Presents football performance data through interactive dashboards.
- Team-wise performance comparison
- Match and scoring trends
- Key performance indicators analysis

---

 4. Global GDP Analytics Dashboard
Visualizes and compares GDP trends across countries.
- Country-wise GDP comparison
- Economic growth trends over time
- Identification of high-performing economies

---

 5. Customer Feedback Analytics Dashboard
Analyzes customer feedback data to measure satisfaction and performance.
- Positive vs negative feedback analysis
- Customer satisfaction trends
- Category-wise feedback insights
